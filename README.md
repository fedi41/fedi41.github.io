# my protfolio website
[CHECK IT OUT](https://fedi41.github.io)

This is a simple website to make my projects easier accessible.

## Website features
- Dynamic logo changing
  - The logo on hthe website changes every few seconds
  - All of the logos are dynamicly loaded from a git repo
  - The images are automaticly passed throw an API to scale them down, and reduce internet usage on mobile
- Dynamic project cards with jekyll
  - All fo the projects are saved in .md files in the _projects repository
  - jekyll loads them, and inserts to the site
  - inspired by hackclubs website
    - backgroound theme, accent color and font can be changed for each project card, just by changing the file in _projects
- Nice design
  - Well I like it )
